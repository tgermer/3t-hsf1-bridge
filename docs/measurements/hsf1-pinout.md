# HSF1 Pinout

## Remote

Battery: CR2450

## Buttons

| Button | Function                              |
| ------ | ------------------------------------- |
| S1     | Extend awning                         |
| S2     | Stop / Favorite position (long press) |
| S3     | Retract awning                        |

## Electrical Test

All buttons were tested by manually shorting PCB contacts.

| Button | Contacts      |
| ------ | ------------- |
| S1     | Pin 1 ↔ Pin 4 |
| S2     | Pin 1 ↔ Pin 4 |
| S3     | Pin 1 ↔ Pin 4 |

Result:

Closing the contacts has exactly the same effect as pressing the button.
